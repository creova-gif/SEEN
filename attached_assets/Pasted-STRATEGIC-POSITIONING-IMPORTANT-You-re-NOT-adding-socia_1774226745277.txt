## SECTION 1 — PROFILE SOCIAL LINKS

Extend user profile schema:

UserProfile {
  userId
  displayName
  bio
  socialLinks: [
    {
      platform: "instagram" | "twitter" | "linkedin" | "youtube" | "tiktok" | "website",
      url
    }
  ]
}

RULES:
• Max 5 links per user
• Validate URLs before saving
• Store only — do not fetch external data

• Use existing profile section
• Show icons or text links inline
• Clicking opens external browser

https://seen.app/profile/{userId}

SHARE CONTENT:
• Name
• Short description
• Link

NO:
• Preview scraping required
• No internal messaging
