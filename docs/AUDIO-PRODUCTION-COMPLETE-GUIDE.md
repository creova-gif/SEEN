### Recording Specifications

**Format:**
- WAV (uncompressed)
- 48kHz sample rate (broadcast standard)
- 24-bit depth
- Mono

**Delivery Format (for platform):**
- MP3 (compressed, for web delivery)
- 320kbps (highest quality MP3)
- Mono
- Normalized to -16 LUFS

**File Naming Convention:**
```
Season2_Story{X}_Chapter{Y}_EN.wav
```

**Examples:**
- `Season2_Story1_Chapter1_EN.wav`
- `Season2_Story1_Chapter2_EN.wav`
- `Season2_Story6_Chapter5_EN.wav`

### Recording Session Plan

**Session 1 (4 hours): Stories 1-2**
- Warm-up: 15 min
- Story 1 (6 chapters × 3-4 min): 18-24 min recording + 30 min retakes = ~1 hour
- Story 2 (6 chapters × 3-4 min): 18-24 min recording + 30 min retakes = ~1 hour
- Break: 15 min
- Review playback and re-record any issues: 1 hour
- Total: ~4 hours

**Session 2 (4 hours): Stories 3-4**
- Same structure as Session 1

**Session 3 (4 hours): Stories 5-6**
- Story 5 (6 chapters): ~1.5 hours
- Story 6 (5 chapters): ~1.5 hours
- Final review: 1 hour

**Total Studio Time:** 12 hours  
**Total Studio Cost:** $1,200-$2,400 (at $100-$200/hour)

### Narrator Direction

**Tone Guidance:**

**For Reflective Stories (1, 3, 5):**
- Calm, contemplative pacing
- Natural pauses between paragraphs
- Emphasis through pacing (not volume)
- Warm, but not sentimental
- Example feel: NPR "This American Life" narrator

**For Urgent Stories (2, 4, 6):**
- Direct, confident delivery
- Tighter pacing (but still clear)
- Emphasis through precision
- Matter-of-fact, but not cold
- Example feel: Investigative journalism narrator

**General Direction:**
- No dramatic character voices
- No emotional manipulation
- Read as if speaking to one person
- Allow silence to carry weight
- Trust the text

### Post-Production (Audio Engineer)

**Per Chapter (35 files):**
1. Remove breaths (keep natural pauses)
2. Remove clicks, pops, background noise
3. Normalize to -16 LUFS (podcast standard)
4. Fade in/out (1 second)
5. Export as WAV + MP3

**Deliverables:**
- 35 WAV files (archival)
- 35 MP3 files (320kbps, for platform)
- Recording log (see below)

**Post-Production Cost:** $500-$1,000 (at $15-$30 per chapter)

## PART 2: AMBIENT AUDIO PRODUCTION

### Composer Brief

**Project:** SEEN by CREOVA — Ambient Soundscapes  
**Purpose:** Underscore narration, create emotional atmosphere  
**Quantity:** 5 unique soundscapes  
**Budget:** $1,500-$4,000 ($300-$800 per soundscape)

### Soundscape 1: Urban Night (Canada)

**Emotional Intent:**
- Quiet city after dark
- Solitude, but not loneliness
- Distant urban hum
- Safety in darkness

**Sonic Elements:**
- Distant traffic (very subtle)
- Occasional footsteps (far away)
- Wind through buildings
- Perhaps: distant jazz from a window
- No sirens, no aggression

**Reference Tracks:**
- Brian Eno "Music for Airports"
- Hildur Guðnadóttir "Chernobyl" (subtle tension)
- Jóhann Jóhannsson (ambient urban)

**Duration:** 4-5 minutes (seamless loop)  
**Assigned To:** Stories 1, 2, 4 (Montreal, Toronto urban narratives)

---

### Soundscape 2: Interior Memory Space

**Emotional Intent:**
- Warm, intimate
- Domestic space
- Nostalgia without sadness
- Home

**Sonic Elements:**
- Wood creaks (very subtle)
- Distant muffled voices (no words)
- Warmth of old building
- Perhaps: ticking clock (minimal)
- No melody, just texture

**Reference Tracks:**
- William Basinski "Disintegration Loops" (memory, decay)
- Max Richter "Sleep" (intimate spaces)

**Duration:** 3-4 minutes (seamless loop)  
**Assigned To:** Story 3 (Black Women's Archive, domestic organizing)

---

### Soundscape 3: Transit & Movement

**Emotional Intent:**
- Motion, journey
- Rails, roads, migration
- Purposeful, not frantic
- Distance traveled

**Sonic Elements:**
- Train rhythms (distant, not literal)
- Low rumble (travel)
- Metal on metal (subtle)
- Wind past windows
- No diesel engines (too aggressive)

**Reference Tracks:**
- Steve Reich "Different Trains"
- Burial (UK electronic, urban transit feel)

**Duration:** 3-4 minutes (seamless loop)  
**Assigned To:** Story 2 (Sleeping Car Porters)

---

### Soundscape 4: Land & Nature (Canada)

**Emotional Intent:**
- Connection to land
- Vastness, space
- Indigenous memory
- Not "wilderness," but presence

**Sonic Elements:**
- Wind through trees
- Water (lake, not ocean)
- Bird calls (Canadian species)
- No drums (avoid cliché)
- No "mystical" flutes

**Reference Tracks:**
- Hildegard Westerkamp (Canadian soundscape composer)
- R. Murray Schafer (Canadian acoustic ecology)

**Duration:** 4-5 minutes (seamless loop)  
**Assigned To:** Story 5 (Africville, connection to land and water)

---

### Soundscape 5: Abstract Cultural Texture

**Emotional Intent:**
- Futurism, possibility
- Not "sci-fi," but forward-looking
- Layered, complex
- Hopeful, but grounded

**Sonic Elements:**
- Synthesized textures
- Processed field recordings (city, voices, music)
- Rhythmic (but not beat-driven)
- Glitch/error as texture
- Digital + analog blend

**Reference Tracks:**
- Arca (experimental electronic)
- Oneohtrix Point Never (abstract textures)
- Holly Herndon (processed voice, future sound)

**Duration:** 2-5 minutes (non-looping acceptable)  
**Assigned To:** Story 6 (Black Canadian Futures, Afrofuturism)

---

## PART 3: AUDIO INTEGRATION (Supabase)

### Storage Structure

**Supabase Buckets:**
```
creova-audio-narration/
  season2/
    story1/
      chapter1_en.mp3
      chapter1_fr.mp3 (after translation)
      chapter1_es.mp3 (after translation)
    story2/
      ...

creova-audio-ambient/
  urban-night.mp3
  interior-memory.mp3
  transit-movement.mp3
  land-nature.mp3
  abstract-texture.mp3
```

### Upload Process

**Step 1: Create Buckets (One-Time)**
```typescript
// In Supabase dashboard or via API
const { data, error } = await supabase.storage.createBucket('creova-audio-narration', {
  public: false,
  fileSizeLimit: 52428800, // 50MB per file
});
```

**Step 2: Upload Files**
```typescript
// For each narration file
const { data, error } = await supabase.storage
  .from('creova-audio-narration')
  .upload('season2/story1/chapter1_en.mp3', file, {
    cacheControl: '3600',
    upsert: false
  });
```

**Step 3: Generate Signed URLs**
```typescript
// Get signed URL (expires in 1 year)
const { data } = await supabase.storage
  .from('creova-audio-narration')
  .createSignedUrl('season2/story1/chapter1_en.mp3', 31536000);

console.log(data.signedUrl); // Use this URL in story metadata
```

**Step 4: Update Story Metadata**
```typescript
// In story data files
export const STORY_BLACK_CANADIAN_RENAISSANCE: ExpandedStoryWorld = {
  chapters: [
    {
      chapterId: 's2-renaissance-ch1',
      audioUrl: {
        en: 'https://[supabase-url]/storage/v1/object/sign/...',
        fr: null, // After FR narration recorded
        es: null, // After ES narration recorded
      },
      ambientAudioUrl: 'https://[supabase-url]/storage/v1/object/sign/urban-night.mp3',
    },
  ],
};
```

### Audio Metadata Schema

```typescript
interface ChapterAudio {
  narrationUrl: {
    en: string;
    fr: string | null;
    es: string | null;
  };
  ambientAudioUrl: string | null;
  duration: number; // seconds
  narrator: string; // "Jane Doe"
  recordedDate: string; // "2026-02-15"
  audioQuality: '320kbps' | '128kbps';
}
```

### Playback Rules

**Narration:**
- Auto-duck ambient audio to -22dB when narration plays
- Resume narration from last position
- Respect user pause/play controls
- No autoplay between chapters (user must advance)

**Ambient:**
- Loop seamlessly
- Fade in (3 seconds) when chapter loads
- Fade out (3 seconds) when chapter changes
- User can disable ambient audio in settings
