## 🌍 MULTILINGUAL PRODUCTION COMPLIANCE

### Language-Specific Requirements

**English (EN):**
- Natural North American pacing
- Clear enunciation without over-articulation
- Neutral accent (Toronto/Vancouver standard)

**French (FR):**
- Quebec French pronunciation standards
- Allow for natural French sentence flow (longer than English)
- Preserve liaison and melodic phrasing
- **DO NOT mimic English rhythm**

**Spanish (ES):**
- Latin American Spanish (neutral variant)
- Adjust for higher syllable density
- Maintain warmth inherent to Spanish phonetics
- Preserve directness in tone-appropriate stories

### Narrator Flexibility

✅ **ALLOWED:** Different narrators per language  
✅ **PRIORITY:** Tone consistency > voice matching  
✅ **RATIONALE:** Cultural authenticity over uniformity

## 📋 TECHNICAL SPECIFICATIONS (CMF Compliance)

### Audio File Requirements

```
FORMAT:       WAV (uncompressed)
SAMPLE RATE:  44.1 kHz or 48 kHz
BIT DEPTH:    16-bit or 24-bit
CHANNELS:     Mono
DURATION:     2-5 minutes (ideal: 3 minutes)
FILE SIZE:    Max 50MB per file
```

### Post-Production Standards

```
NORMALIZATION:  LUFS -16 (broadcast standard)
NOISE FLOOR:    Below -60dB
PEAK LEVEL:     Max -3dB (no clipping)
FADE IN:        0.5 seconds
FADE OUT:       1.0 seconds
```

### File Naming Convention

```
Format: StoryWorld_ChapterID_Language.wav

Examples:
  MidnightResonance_Ch1_EN.wav
  SoftPower_Ch3_FR.wav
  SeenUnseen_Ch2_ES.wav
```
