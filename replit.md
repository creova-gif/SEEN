# SEEN by Creova

## Overview
A React-based frontend application called "SEEN" by Creova. Built with Vite, Tailwind CSS v4, and a rich set of UI libraries including Radix UI, MUI, and Recharts.

## Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS v4 with @tailwindcss/vite plugin
- **UI Libraries**: Radix UI, MUI Material, Lucide React icons
- **Animation**: Motion (framer-motion successor)
- **Charts**: Recharts
- **Drag & Drop**: react-dnd

## Project Structure
- `src/main.tsx` - Entry point
- `src/app/App.tsx` - Root application component
- `src/app/components/` - Reusable UI components
- `src/app/contexts/` - React context providers
- `src/app/data/` - Data files and constants
- `src/app/hooks/` - Custom React hooks
- `src/app/navigation/` - Navigation/routing
- `src/app/types/` - TypeScript type definitions
- `src/app/utils/` - Utility functions
- `src/styles/` - Global styles

## Development
- Dev server: `npm run dev` (port 5000)
- Build: `npm run build`

## Deployment
- Static deployment using `dist/` directory after build
